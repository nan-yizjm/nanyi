# 518.零钱兑换II
```cpp
class Solution {
public:
    int change(int amount, vector<int>& coins) {
        vector<unsigned long long>dp(amount+2);
        dp[0]=1;
        for(int i=0;i<coins.size();i++)
            for(int j=coins[i];j<=amount;j++)
                dp[j]=dp[j]+dp[j-coins[i]];
        return dp[amount];
    }
};
```

这道题由动态规划五部曲分析：
- 确定dp[i]含义:dp[i]表示组成金额i的组合数。
- 递推公式:dp[i]=dp[i]+dp[i-coins[j]],其中coins[j]表示当前考虑的硬币面额。
- dp数组初始化:dp[0]=1,其余为0。
- 遍历顺序:先遍历硬币，再遍历金额(正序，因为每个硬币可以使用多次)。
- 打印dp[i]的值
