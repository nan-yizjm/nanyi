# 1049.最后一块石头的重量II
```cpp
class Solution {
public:
    int lastStoneWeightII(vector<int>& stones) {
        if(stones.size()==1) return stones[0];
        int sum=0;
        for(int i=0;i<stones.size();i++)
            sum+=stones[i];
        int n=sum/2;
        vector<int> dp(n+2);
        for(int i=0;i<=n;i++)
            dp[i]=0;
        for(int i=0;i<stones.size();i++)
            for(int j=n;j>=stones[i];j--)
                dp[j]=max(dp[j],dp[j-stones[i]]+stones[i]);
        int result=0;
        for(int i=n;i>=0;i--)
            if(dp[i]!=0)
            {
                result=sum-dp[i]*2;
                break;
            }
        return result;
    }
};
```

这道题由动态规划五部曲分析：
- 确定dp[j]含义:dp[j]表示容量为j的背包最大能装的石头重量总和,将石头重量视为物品重量,最后两堆石头的重量差就是答案。
- 递推公式:dp[j]=max(dp[j],dp[j-stones[i]]+stones[i])
- dp数组初始化:dp[j]=0
- 遍历顺序:先遍历物品，再遍历背包容量（从后往前）。
- 打印dp[j]的值
