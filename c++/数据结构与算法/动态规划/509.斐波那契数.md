# 509.斐波那契数
```cpp
class Solution {
public:
    int fib(int n) {
        if(n<=1) return n;
        vector<int> dp(n+1);
        dp[0]=0;
        dp[1]=1;
        for(int i=2;i<=n;i++)
            dp[i]=dp[i-1]+dp[i-2];
        return dp[n];
    }
};
```

状态压缩
```cpp
class Solution {
public:
    int fib(int n) {
        if(n<=1) return n;
        int dp0=0,dp1=1,sum=0;
        for(int i=2;i<=n;i++)
        {
            sum=dp0+dp1;
            dp0=dp1;
            dp1=sum;
        }
        return sum;
    }
};
```

这道题由动态规划五部曲分析：
- 确定dp[i]含义:dp[i]表示斐波那契数列的第i项的值。
- 递推公式:dp[i]=dp[i-1]+dp[i-2]。
- dp数组初始化:dp[0]=0,dp[1]=1。
- 遍历顺序:从前往后遍历。
- 打印dp[i]的值
