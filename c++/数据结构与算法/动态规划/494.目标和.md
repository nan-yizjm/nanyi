# 494. 目标和
```cpp
class Solution {
public:
    int findTargetSumWays(vector<int>& nums, int target) {
        int sum=0;
        for(int i=0;i<nums.size();i++)
            sum+=nums[i];
        if((sum+target)%2==1||(sum+target)<0) return 0;
        int n=(sum+target)/2;
        vector<int>dp(n+1);
        dp[0]=1;
        for(int i=1;i<=n;i++)
            dp[i]=0;
        for(int i=0;i<nums.size();i++)
            for(int j=n;j>=nums[i];j--)
                dp[j]=dp[j]+dp[j-nums[i]];
        return dp[n];
    }
};
```

这道题由动态规划五部曲分析：
- 确定dp[j]含义:dp[j]表示容量为j的背包有多少种装法，n=(sum+target)/2表示背包的容量，即将数组分为正数和负数两部分，使得正数部分的和减去负数部分的和等于target。
- 递推公式:dp[j]=dp[j]+dp[j-nums[i]]
- dp数组初始化:dp[0]=1, dp[1..n]=0
- 遍历顺序:先遍历物品，再遍历背包容量（从后往前）。
- 打印dp[j]的值
