# 343.整数拆分
```cpp
class Solution {
public:
    int integerBreak(int n) {
        vector<int> dp(n+2);
        dp[0]=0;
        dp[1]=0;
        dp[2]=1;
        for(int i=3;i<=n;i++)
            for(int j=1;j<=i;j++)
                dp[i]=max(j*(i-j),max(j*dp[i-j],dp[i]));
        return dp[n];
    }
};
```

这道题由动态规划五部曲分析：
- 确定dp[i]含义:dp[i]表示将整数i拆分的最大乘积。
- 递推公式:dp[i]=max(j*(i-j),max(j*dp[i-j],dp[i])),其中j表示拆分的第一个整数，i-j表示剩余的整数。第一种情况是直接将i拆分成j和i-j，第二种情况是将i拆分成j和dp[i-j]。
- dp数组初始化:dp[0]=0,dp[1]=0,dp[2]=1。
- 遍历顺序:从小到大遍历。
- 打印dp[i]的值
