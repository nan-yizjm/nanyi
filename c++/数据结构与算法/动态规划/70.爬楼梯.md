# 70.爬楼梯
```cpp
class Solution {
public:
    int climbStairs(int n) {
        if(n<=2) return n;
        vector<int> dp(n+1);
        dp[1]=1;
        dp[2]=2;
        for(int i=3;i<=n;i++)
            dp[i]=dp[i-1]+dp[i-2];
        return dp[n];
    }
};
```

这道题由动态规划五部曲分析：
- 确定dp[i]含义:dp[i]表示爬到第i个台阶的方法数。
- 递推公式:dp[i]=dp[i-1]+dp[i-2]。
- dp数组初始化:dp[1]=1,dp[2]=2,题目要求n>0,所以dp[0]无意义,所以从dp[1],dp[2]开始。
- 遍历顺序:从前往后遍历。
- 打印dp[i]的值
