# 474.一和零
```cpp
class Solution {
public:
    int findMaxForm(vector<string>& strs, int m, int n) {
        vector<vector<int>>dp(m+1,vector<int>(n+1));
        for(int i=0;i<=m;i++)
            for(int j=0;j<=n;j++)
                dp[i][j]=0;
        for(int i=0;i<strs.size();i++)
        {
            int sum0=0,sum1=0;
            for(char c:strs[i])
            {
                if(c=='0') sum0++;
                else sum1++;
            }
            for(int j=m;j>=sum0;j--)
                for(int k=n;k>=sum1;k--)
                    dp[j][k]=max(dp[j][k],dp[j-sum0][k-sum1]+1);
        }
        return dp[m][n];
    }
};
```

这道题由动态规划五部曲分析：
- 确定dp[i][j]含义:dp[i][j]表示i个0和j个1能组成的字符串子集个数的最大数量。
- 递推公式:dp[i][j]=max(dp[i][j],dp[i-sum0][j-sum1]+1)，其中sum0和sum1分别是当前字符串中0和1的数量。
- dp数组初始化:dp[i][j]=0
- 遍历顺序:先遍历字符串，再遍历背包容量（从后往前）。
- 打印dp[i][j]的值
