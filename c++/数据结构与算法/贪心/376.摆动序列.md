# 376.摆动序列
```cpp
class Solution {
public:
    int wiggleMaxLength(vector<int>& nums) {
        if(nums.size()<=1) return nums.size();
        int pre=0;
        int cur;
        int sum=1;
        for(int i=0;i<nums.size()-1;i++)
        {
            cur=nums[i+1]-nums[i];
            if((pre>=0&&cur<0)||(pre<=0&&cur>0))
            {
                sum++;
                pre=cur;
            }
        } 
        return sum;
    }
};
```

这道题注意对情况的全面考虑，第一上下坡有平坡，第二全单调，第三单调平坡单调。本题通过将pre=0处理首元素，等于复制一个首元素的平坡，同时pre不用实时更新，只需要在变方向时更新即可。
