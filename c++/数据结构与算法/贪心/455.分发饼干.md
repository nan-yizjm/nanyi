# 455. 分发饼干
```cpp
class Solution {
public:
    int findContentChildren(vector<int>& g, vector<int>& s) {
        sort(g.begin(),g.end());
        sort(s.begin(),s.end());
        int k=0;
        for(auto it=s.begin();it!=s.end()&&k<g.size();++it)
            if(*it>=g[k])
                k++;
        return k;
    }
};
```

这道题利用小饼干尽量满足小胃口的思路，从而尽可能满足更多孩子。

### 知识补充
- `sort`函数：用于对容器中的元素进行排序，默认使用升序排序。
- **lambda表达式**：一种匿名函数，可以直接在需要函数对象的地方定义和使用。
$$[捕获列表](参数列表) mutable 异常规范 -> 返回类型 { 函数体 }$$

| 捕获方式 | 语法 | 说明 |
| :--- | :--- | :--- |
| 不捕获 | `[]` | 不访问任何外部变量。 |
| 值捕获 | `[x]` | 创建外部变量 `x` 的一个副本。 |
| 引用捕获 | `[&x]` | 直接引用外部变量 `x`，修改会影响原变量。 |
| 隐式值捕获 | `[=]` | 按值捕获所有在 Lambda 中使用到的外部变量。 |
| 隐式引用捕获 | `[&]` | 按引用捕获所有在 Lambda 中使用到的外部变量。 |
| 混合捕获 | `[=, &x]` | 大部分变量按值捕获，但 `x` 按引用捕获。 |
