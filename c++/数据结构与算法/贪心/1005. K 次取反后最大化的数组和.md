# 1005. K 次取反后最大化的数组和
```cpp
class Solution {
public:
    int largestSumAfterKNegations(vector<int>& nums, int k) {
        sort(nums.begin(),nums.end(),[](int a,int b){return std::abs(a)>std::abs(b);});
        for(int i=0;i<nums.size();i++)
        {
            if(nums[i]<0&&k>0)
            {
                nums[i]=-nums[i];
                k--;
            }
        }
        if(k%2==1) nums[nums.size()-1]=-nums[nums.size()-1];
        int sum=0;
        for(int i=0;i<nums.size();i++)
            sum=sum+nums[i];
        return sum;
    }
};
```

这道题通过两次贪心达到全局最优解，第一次贪心是对于绝对值更大的负数优先取反，第二次贪心是当k还有剩余时，对于绝对值最小的正数取反，通过对k的奇偶性判断是否需要对绝对值最小的数取反。
