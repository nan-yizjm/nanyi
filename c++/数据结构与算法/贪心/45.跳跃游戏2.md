# 45.跳跃游戏2
```cpp
class Solution {
public:
    int jump(vector<int>& nums) {
        if(nums.size()<=1) return 0;
        int cur=0,next=0;
        int sum=0;
        for(int i=0;i<nums.size();i++)
        {
            next=max(nums[i]+i,next);
            if(i==cur)
            {
                if(cur!=nums.size()-1)
                {
                    cur=next;
                    sum++;
                    if(cur>=nums.size()-1) break;
                }
                else break;
            }
        }
        return sum;
    }
};
```

这道题仍是通过覆盖范围进行判断的，但为了记录最少次数，需要通过cur和next进行判定何时增加次数。cur记录当前覆盖范围的最远位置，next记录下一次覆盖范围的最远位置，当i达到cur时，说明需要增加一次跳跃了，此时将cur更新为next，并增加次数sum。最后返回sum即可。
