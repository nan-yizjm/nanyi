# 53.最大子数组和
```cpp
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int result=INT_MIN;
        int count=0;
        for(int i=0;i<nums.size();i++)
        {
            count+=nums[i];
            if(count>result) result=count;
            if(count<0) count=0;
        }
        return result;
    }
};
```

这道题考虑局部最优的贪心算法是考虑连续和而不是单个元素的正负值，如果当前连续和小于0了，那么就不可能对后续的元素有正面影响了，所以直接把连续和置0，重新开始计算连续和。但是单个元素可能后面元素可以带来正面影响。
