# 134.加油站
```cpp
class Solution {
public:
    int canCompleteCircuit(vector<int>& gas, vector<int>& cost) {
        int cursum=0,totalsum=0,start=0;
        for(int i=0;i<gas.size();i++)
        {
            cursum+=(gas[i]-cost[i]);
            totalsum+=(gas[i]-cost[i]);
            if(cursum<0)
            {
                start=i+1;
                cursum=0;
            }
        }
        if(totalsum<0) return -1;
        return start;
    }
};
```

这道题将问题转化为每个站点的净油量（gas[i] - cost[i]）。我们维护一个当前的净油量（cursum）和总的净油量（totalsum）。如果在某个站点cursum变为负数，说明从之前的起点到这个站点无法完成行程，我们将起点更新为下一个站点，并重置cursum为0。最后，如果totalsum小于0，说明无论从哪个站点出发都无法完成行程，返回-1；否则返回起点。通过只关心每一站的局部最优解，我们能够在一次遍历中找到全局最优解。
