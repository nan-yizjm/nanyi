# 135.分发糖果
```cpp
class Solution {
public:
    int candy(vector<int>& ratings) {
        vector <int> candy(ratings.size(),1);
        for(int i=1;i<ratings.size();i++)
            if(ratings[i]>ratings[i-1])
                candy[i]=candy[i-1]+1;
        for(int i=ratings.size()-2;i>=0;i--)
            if(ratings[i]>ratings[i+1])
                candy[i]=max(candy[i+1]+1,candy[i]);
        int sum=0;
        for(int i=0;i<ratings.size();i++)
            sum=sum+candy[i];
        return sum;
    }
};
```

这道题的核心思想是：先从左到右遍历一次，确保每个孩子比左边的孩子评分高时糖果数比左边多；再从右到左遍历一次，确保每个孩子比右边的孩子评分高时糖果数比右边多。最后将所有孩子的糖果数相加得到结果。从两边分别遍历，用max处理冲突的情况，保证每个孩子的糖果数满足条件。
