# 55.跳跃游戏
```cpp
class Solution {
public:
    bool canJump(vector<int>& nums) {
        int cover=0;
        for(int i=0;i<=cover;i++)
        {
            cover=max(i+nums[i],cover);
            if(cover>=nums.size()-1) return true;
        }
        return false;
    }
};
```

这道题通过将跳跃的方案转化为范围的覆盖问题，只要能够覆盖就说明可以到达该位置。
