### 🗺️ 一、LLM全栈学习路线（5大阶段）

该路线旨在覆盖从基础理论到工业级系统开发的全过程，总周期根据个人投入有所不同。

| 阶段 | 核心目标 | 关键内容与产出 |
| :--- | :--- | :--- |
| **阶段1：基础打底**(0–3个月) | 看懂论文、跑通代码、理解Transformer | **内容**：Python/PyTorch、深度学习基础（CNN/RNN）、数学基础、Transformer原理。**产出**：手写简易Transformer，跑通GPT-2/BERT。 |
| **阶段2：LLM核心原理**(2–4个月) | 理解模型为何有效、训练与推理机制 | **内容**：主流架构（GPT/LLaMA）、预训练目标（Causal LM/MLM）、位置编码（RoPE/ALiBi）、推理优化（KV Cache）。**产出**：复现mini-GPT，解释各层作用。 |
| **阶段3：工程与训练**(3–6个月) | 训练、微调、部署可用的LLM | **内容**：高效微调（LoRA/QLoRA）、分布式训练（FSDP/DeepSpeed）、对齐（SFT/DPO）、部署（vLLM/量化）、RAG。**产出**：微调Llama-2-7B，开发对话/问答应用。 |
| **阶段4：进阶与前沿**(持续学习) | 做研究、创新、复杂系统开发 | **内容**：新架构（MoE/Mamba）、多模态（LLaVA）、Agent、高级对齐（RLHF）、安全与红队。**产出**：复现顶会论文，构建多模态/Agent系统。 |
| **阶段5：研究与系统**(长期) | 独立科研、工业级系统设计 | **方向**：模型架构改进、高效训练、安全对齐、大模型系统工程（分布式/云原生）。 |

### ✅ 二、阶段自测标准（掌握度 >80%）

文档提供了详细的Checklist，用于判断是否具备进入下一阶段的能力。

#### 1. 阶段1：基础打底（工程能力）
- **Python/数学**：熟练使用Numpy/Pandas，掌握线性代数与微积分基础。
- **PyTorch**：能写训练循环，理解Autograd，会定义`nn.Module`。
- **模型基础**：能画出Transformer结构图，解释自注意力公式，手写简易Transformer。
- **合格标志**：能看懂、跑通、修改、解释Transformer与基础深度学习代码。

#### 2. 阶段2：核心原理（理论深度）
- **架构理解**：透彻理解LLaMA/Mistral架构（RMSNorm, RoPE, SwishGLU），区分Encoder/Decoder模式。
- **核心机制**：解释Causal LM与MLM区别，理解KV Cache加速原理，掌握各类采样策略。
- **合格标志**：能清晰对比RoPE vs ALiBi、Pre-Norm vs Post-Norm，并解释SFT原理。

#### 3. 阶段3：工程与训练（落地能力）
- **微调与部署**：独立跑通LoRA/QLoRA微调，使用vLLM/TGI部署，掌握GPTQ/AWQ量化。
- **系统构建**：搭建RAG系统（分块、检索、重排），配置FSDP/DeepSpeed进行分布式训练。
- **合格标志**：能独立完成“数据清洗 → LoRA微调 → 量化部署 → RAG应用”的全流程。

#### 4. 阶段4：进阶与前沿（创新能力）
- **前沿技术**：理解MoE（混合专家）、Mamba（状态空间模型）等Transformer替代架构。
- **复杂系统**：构建包含工具调用、反思能力的Agent，实现多模态图文问答。
- **研究能力**：复现顶会论文（如DPO/Mamba），进行模型安全红队测试。
- **合格标志**：具备研究思维，能跟踪前沿并构建如“Agent+RAG+多模态”的复杂系统。
